# sources

Raw, immutable inputs go here: articles, notes, transcripts, papers, exported
docs, whatever you want the wiki to learn from. Add new files; don't edit or
delete existing ones as part of a normal wiki workflow (if a source itself
was wrong, that's a rare, deliberate correction — not routine maintenance).

Run `/ingest <path>` (or `/ingest <pasted text>`) to have the agent fold a
source into `../wiki/`.
