CXL(Compute Express Link)은 CPU-메모리 간 인터커넥트 표준으로, 메모리
풀링·가상화(메모리를 서버 간에 유연하게 공유·확장)를 가능케 한다.
사용자가 개인적으로 관심을 갖는 주제이자, SK하이닉스 밸류에이션에서
HBM 이후의 "다음 성장 엔진"으로 주목하는 트랙이라 별도 페이지로 관리한다.
[체크포인트 프레임워크](sk-hynix-analyst-thesis-checkpoints.md)의 9개
항목과는 별개로, 매일 리포트 맨 아래 "Deep Dive" 특별 코너로 독립
추적한다.

## 왜 추적하나

- HBM은 이미 시장이 가격에 반영한 서사인 반면, CXL은 아직 초기 단계 —
  선반영 여부를 판단할 수 있는 "다음 촉매" 후보
- 표준화 진행 속도, 하이퍼스케일러의 실제 채택(AI 인프라의 KV-cache
  메모리 절감 등 구체 유스케이스), 경쟁사(삼성·마이크론) 로드맵, 학계
  연구 흐름을 종합해야 실제 상용화 임박도를 가늠할 수 있음
- SK하이닉스의 CXL 컨트롤러/모듈 자사화 진행률이 HBM 이외의 두 번째
  차세대 라인업으로 가시화되는지가 중기 멀티플(P/E) 상향 요인이 될 수
  있다는 것이 사용자의 관심 축

## 현재 상태 (2026-07-24 첫 체크 — 미리보기 데모로 채움)

### ⓐ CXL 표준화 & AI 산업 빅이벤트
- **[FACT] CXL 4.0 스펙 공식 발표(2025-11-18, CXL Consortium)**: 대역폭
  64GT/s→128GT/s로 2배 확대(PCIe 7.0 연계), Bundled Ports 지원으로 최대
  1.5TB/s 연결 가능, 메모리 RAS 기능 강화. 단, 2026년 현재 상용 실리콘은
  대부분 CXL 2.0 수준이고 CXL 4.0 멀티랙 시스템은 2026년 말~2027년 배포
  전망 — 스펙 발표와 실제 상용화 사이 시차가 여전히 큼.
- **[FACT] Microsoft Azure 업계 최초 상용 CXL 메모리 배포**(2025-11): M-series
  VM(Intel Xeon 6 + Astera Labs Leo CXL 컨트롤러) — 스펙 단계를 넘어 실제
  클라우드 서비스 단계 진입.
- **[FACT] Penguin Solutions, CXL 기반 KV-cache 서버 양산 출시**(2026-03,
  최대 11TB, NVMe 대비 10배 속도) — LLM 추론 인프라에서 CXL이 실질적
  해법으로 자리잡는 구체 사례.
- **[OPINION] TCO 절감 추정**: CXL 기반 아키텍처가 하이퍼스케일러 총소유비용을
  15~20% 절감한다는 업계 추정(Introl 등) — 채택 유인은 강하나 아직
  업계 전반 확산 단계는 아님.

### ⓑ 최신 연구/논문 및 학술 트랙
- **[FACT] CXL-SpecKV**(2026, arXiv 2512.11920): CXL 메모리 디스어그리게이션
  + FPGA 추측실행 결합 — KV-cache 4~8배 용량 확장, 95% 정확도 프리페칭으로
  지연시간 감소.
- **[FACT] HyMCache**(2026, arXiv 2607.18141): CXL-하이브리드 메모리 기반
  멀티턴 LLM 서빙 프레임워크 — SSD급 비용으로 DRAM급 KV-cache 효율 지향.

### ⓒ 경쟁사 뉴스룸 및 수급 동향
- **[FACT] 삼성전자 CMM-D 3.0**(CXL 3.2 기반): 2026년 말 양산 목표(TrendForce,
  2026-07-21). CXL 3.2의 CHMU(Hot Page Monitoring Unit)로 데이터 접근빈도
  실시간 모니터링·배치 최적화 기능 추가. 앞서 CXL 3.1 기반 버전은 인텔·AMD
  CPU 출시 지연으로 양산이 한 차례 연기된 바 있음.
- **[FACT] 마이크론**: CZ120(CXL2.0, 128/256GB) 출하 중, 차세대 CZ122
  샘플링 중(하드웨어 기반 이종 인터리빙 추가, Red Hat 인증).
- **[FACT] ⚠ 3사(삼성·SK하이닉스·마이크론) 공통 — 자체 CXL 컨트롤러 IC
  상용화 계획 전략적 중단**: 데이터센터 고객들이 통합형 솔루션보다 범용
  메모리 모듈+독립 컨트롤러 조합을 선호하는 것으로 확인(BigGo Finance
  보도). "컨트롤러 자사화"가 오히려 역행하는 흐름이라는 점이 이번 체크의
  가장 눈에 띄는 발견.

### ⓓ [FACT vs OPINION] SK하이닉스 CXL 경쟁력 & 주가 영향
- **[FACT] 2세대 CMM-DDR5 공개**(2026년 6월, HPE Discover): CXL 3.2 기반
  256GB — 1세대(CXL2.0+, 128GB) 대비 세대 내 용량 2배. 고객 공급 논의
  진행 중이나 양산 시점은 미공개.
- **[FACT] IMTE(Inference Memory Tiering Expansion) 아키텍처**: HBM/DDR와
  SSD 사이에 CXL 하이브리드 메모리를 배치해 AI 추론 효율을 기존 대비
  35.7% 개선(자체 발표치, 제3자 검증 미확인).
- **[FACT] SK하이닉스도 자체 CXL 컨트롤러 IC 상용화 계획 중단**(위 ⓒ와
  동일한 3사 공통 전략 선회).
- **[OPINION] 밸류에이션 반영은 아직 이름**: Seoul Economic Daily(7/19~20)는
  삼성·SK하이닉스의 CXL 3.2 양산 경쟁을 "CXL 메모리 전쟁"으로 표현할 만큼
  본격화됐다고 평가하나, 아직 매출 기여는 초기 단계 — 밸류에이션(P/E)
  상향 요인으로 반영되기엔 이르다는 게 현재 컨센서스에 가까움(9개
  체크포인트에는 아직 미포함). "HBM 이후 두 번째 성장엔진"으로 시장에
  본격 인식되려면 실제 양산 시점 공식화가 먼저 확인돼야 함 — 다음
  체크에서 계속 추적.
