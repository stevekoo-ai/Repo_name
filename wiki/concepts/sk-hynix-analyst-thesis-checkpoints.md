훼손되는지 확인하기 위한 체크리스트. 사용자가 제공한 11개 기관 리포트
요약본(엑셀)과 웹 검색을 교차 검증해 구성했다. 새 뉴스·차트를 `/ingest`
할 때마다 아래 항목의 상태를 갱신한다. 판단 근거일 뿐 투자 조언이 아니다.

매일 세 번, 이 페이지를 자동으로 점검·갱신하는 Routine이 등록되어 있다:

  살아있는지 점검, 스코어보드 갱신. 장 시작 전이라 당일 개장 반응은 못 잡음
- **장초반 10:00(KST, 2026-07-16 신설)** — 09~10시 개장 직후 움직임과 원인을
  확인하는 속보 성격. 최근 며칠 중요한 움직임이 전부 개장 직후 나왔던 것에
  착안해 추가(trig_015besxPhzkuUSDbQa6AkmgD). 특이사항 없어도 매일 발행
- **저녁 19:00(KST)** — 장마감 후, 새로 나온 애널리스트 목표가·리포트 발굴 +

**2026-07-24 갱신 — "SK하이닉스 Daily Monitor" 포맷으로 리포트 구조 전면

변경점:
1. **제목 형식**: `[SK하이닉스 Daily Monitor] [아침/장초반/저녁] - YYYY-MM-DD HH:MM`
2. **섹션 2 신설 — 🌐 하이퍼스케일러 고객 동향(CAPEX·메모리수요)**: 기존
   "추적 프레임 A(경쟁생존논리 vs ROIC규율)"를 이 섹션으로 흡수·확장.
   - 🇺🇸 미국(Google·MS·Amazon·Meta): [FACT] 핵심 발표 + [OPINION] SK하이닉스
     영향 해석 — "경쟁생존논리 vs ROIC규율" 프레임은 이 OPINION 파트에 계속 적용
   - 🇨🇳 중국(ByteDance·Alibaba·Tencent·Baidu): [FACT]+[OPINION] — **신규
     추적 축**. 기존엔 CXMT(공급측 경쟁자)만 추적했는데, 중국 하이퍼스케일러의
     범용 DRAM/NAND 수요(HBM은 수출규제로 제외)라는 별개의 수요측 앵글을
     새로 추가. 체크포인트 ⑦(CXMT)과는 구분해서 관리 — ⑦은 공급 위협,

3. **FACT/OPINION 명시적 라벨링 도입**: 9개 체크포인트 요약표에도 "확인된
   사실"과 "전문가/IB 전망"을 구분하는 관행을 적용 — 검색 결과의 확정
   사실과 애널리스트 추정을 혼동하지 않는다는 기존 원칙("숫자를 지어내지
   말 것")을 표 구조로 명문화.
4. 찐반등 4대 신호, 한줄진단 등 기존 섹션은 유지, 관전포인트는 한줄진단과

**2026-07-24 갱신 #4 — HBM Cycle Score & 선행지표 조기경보(HSEWS) 도입**:
사용자가 제시한 "HBM Supercycle Early Warning System" 설계를 검토·
반영. 전용 페이지 [hbm-cycle-score.md](hbm-cycle-score.md) 신설. 핵심
결정 3가지(모두 사용자 확인):
1. **종합점수는 기존 구조에 얹는 요약 레이어** — 9개 체크포인트 스코어보드·
   찐반등4대신호는 그대로 유지, HBM Cycle Score(0~100)+가설 붕괴조건(0/4)을
   리포트 최상단에 신규 섹션으로 추가
2. **실행 빈도는 하루 3회 유지** — 매시간 실행 제안은 기각(웹검색 소스가
   시간 단위로 갱신되지 않아 비용 대비 이득 작음)
3. **엔비디아 주문·CoWoS 활용률은 독립 Tier-1 섹션으로 신설** — 하이퍼스케일러
   섹션(CapEx 위주)과 분리, "CoWoS→엔비디아주문→HBM발주"로 이어지는
   한 단계 더 앞선 선행지표라는 논리 반영

부수 변경: 체크포인트①에 "HBM ASP 3주 연속 하락 = 경고" 임계치 규칙 추가,
market-cycles-leverage-risk.md에 SK하이닉스 종목 기준 외국인 순매수
1/5/20/60일 누적+보유율 델타 정량 추적("1-4-1") 신설, 하이퍼스케일러
고객 동향 섹션에 고객사별 재고 센티먼트 점수(0~10) 하위 필드 추가.
리포트 섹션 순서(아침·저녁 풀버전): 1.주가현황 → 2.🚨HBM Cycle
Score+붕괴조건 → 3.⚡엔비디아주문&CoWoS → 4.🌐하이퍼스케일러 →
5.체크포인트매트릭스 → 6.찐반등4대신호 → 7.CXL Deep Dive → 8.한줄진단.
장초반은 전 섹션 간략 버전으로 포함(생략 없음).

**2026-07-24 갱신 #5 — 거시국면(G/I/L) & 역사적 유사 시기 레이어 신설**:
사용자가 "매크로 국면 Daily 리포트" 구현요청서 제시(원래는 FRED
API+cron 기반 별도 Python 프로젝트 스펙) → 지금 구조(위키+웹검색+
Routine)에 맞게 재해석해 흡수. 전용 페이지
[macro-regime-history.md](macro-regime-history.md) 신설, 계산 엔진은
[scripts/regime_engine.py](../../scripts/regime_engine.py)로 구현(라우틴이
bash로 직접 실행). 데이터 소스는 사용자가 이미 다른 세션에서 운영 중인
FRED API 파이프라인 산출값과 이 라우틴의 웹검색 확인, 둘 다 같은
`--append-month` 경로로 주입 가능하게 설계 — 어느 쪽으로 넣어도 동일하게
작동. **배치 빈도(사용자 확인 없이 기본값으로 진행, 이견 있으면 조정)**:
G/I/L은 월 1회만 실제로 바뀌는 느린 지표라 **아침 1회만 풀버전**, 장초반·
저녁은 국면 변화가 있을 때만 한 줄 요약. 리포트 섹션 순서에 "0. 🌐
오늘의 매크로 헤드라인"으로 최상단(HBM Cycle Score보다 위) 배치 — SK하이닉스
미시 신호보다 먼저 거시 배경을 보여주는 게 사용자가 원한 "이야기 흐름"에
맞음. SK하이닉스 자체의 "이격도(divergence) 게이지"(주가지수 vs
영업이익지수 로그차)는 [rally-justification-analysis.md](rally-justification-analysis.md)에
신설 — 분기별 정밀 시계열은 아직 없어 24Q1→26Q1 2점 비교 근사치만 우선
반영, 사용자가 보유한 분기별 주가 데이터(`hynix-hbm-rally.jsx` 언급)를
받으면 정밀화 예정.

위험↑/위험↓/상승기대↑/상승기대↓)를 아침 전용에서 장초반·저녁까지

거시 이벤트가 있을 때만 카드 위에 한 줄 추가, 저녁은 그날 확인/반박된
지표가 있으면 체크리스트(⏳→✅/❌)와 수렴도만 갱신하는 "하루 마감
확정판" 역할.

각 실행마다 위키를 갱신·커밋하고, HTML 리포트를 Artifact로 재발행(고정 URL)한
뒤 채팅 요약 + 푸시 알림을 보낸다. Routine 자체는 이 위키 스키마 밖의
인프라(계정에 연결된 스케줄러)라 CLAUDE.md에는 기록하지 않지만, 이 페이지가
자동 갱신 대상이라는 사실은 여기 남겨둔다.

발행된 리포트 자체도 매번 `sources/`에 원문 그대로 보관한다 — 위키가 만든
산출물이 다시 다음 갱신의 원자료(raw)로 순환되는 구조(예:
[sources/sk-hynix-morning-report-2026-07-15.html](../../sources/sk-hynix-morning-report-2026-07-15.html),
[sources/sk-hynix-evening-report-2026-07-15.html](../../sources/sk-hynix-evening-report-2026-07-15.html)).
이렇게 하면 나중에 "그날 리포트가 실제로 뭐라고 했는지"를 위키 서술과
별개로 원문 그대로 다시 확인할 수 있다.

### ✅ 해결 — ADR 프리미엄 계산 완료 (2026-07-16 아침 체크)

그동안 막혀 있던 ADR 비율이 확인됐다: **1 ADR = 본주 1/10주**. 이를 바탕으로
7/15 기준 실제 계산 — SKHY $182.6 vs 본주 2,082,000원(≈$1,397, 원/달러
약 1,490) → 1/10 환산 시 본주 쪽 $139.7 → 프리미엄 = (182.6-139.7)/139.7
**≈ 30.7%**.

판정 기준(10~25% 안정 / 25~35% 높음 / 35%+ 과열·훼손)에 대입하면
**"높음·중립" 구간**이며, 7/14 한때 찍었던 정점(약 51.5%, 훼손권)에서
빠르게 정상화(narrowing)되는 방향이다. 즉 지난 며칠간 우려했던 "이미
훼손 구간"이라는 가설은 기각 — 결제·환전 등 신규 상장 초기 마찰 요인이
완화되며 프리미엄이 좁혀지고 있다. 상세는
[sources/websearch-sk-hynix-morning-check-2026-07-16.md](../../sources/websearch-sk-hynix-morning-check-2026-07-16.md).

## 체크포인트 → 목표가 시나리오 매핑

9개 체크포인트를 하나하나 뜯어보면, 실제로 강세·약세 진영이 **정반대로
해석하는 항목은 2개(③ 빅테크 CapEx, ⑥ ADR 프리미엄)뿐**이고, 나머지 7개는
정도 차이는 있어도 대체로 같은 방향(강세)을 가리킨다. 즉 이 2개가 최종
목표가가 강세 클러스터(380만~420만원, Barclays $330)로 수렴할지, 약세
클러스터(BNK 185만원, Morningstar $160)로 수렴할지를 가르는 **진짜 분기점**이고,
나머지 7개는 "강세론의 기본 전제가 아직 살아있는지"를 확인하는 컨센서스
확인용에 가깝다.

## 핵심 체크포인트 (교차 검증)

각 체크포인트를 지지/반박하는 기관과, 엑셀이 제시한 판정 기준을 함께 정리.

### 1. DRAM/NAND 가격 모멘텀
- 기준: 상승 지속(긍정) / 상승률 둔화(중립) / 하락 전환(훼손)
- **경고 규칙(2026-07-24 추가, [HBM Cycle Score](hbm-cycle-score.md) 연동)**:
  HBM ASP가 주간 기준 **3주 연속 하락**하거나, 분기 ASP 가이던스가
  하향되면 즉시 훼손(빨간 신호)으로 격상. 지금까지는 방향성만 판정했는데,
  이 규칙부터 주간 %수치를 명시적으로 기록해 임계치 판단에 사용.
- 지지: KB(2027 공급 7%/4% vs 수요 17%/19%), IBK, NH, 신한(GPM 80% 유지 전제)
- **(2026-07-23 낮 인제스트) 분기별 정량 데이터 + 공급충분성 전망**: D5 RDIMM
  서버 ASP C3Q26 +20~25% QoQ(~$2.4/Gb), 서버수요 ≥65%YoY vs 공급 ~40~45%YoY.
  NAND DC/eSSD ASP +25~30% QoQ, 수요 >80%YoY vs 출하 <50%YoY(20%p+ 갭).
  **공급 충분성(하이퍼스케일 기준) CY26 약 70~80% → CY27 최저 60%까지 악화
  가능 — CY27이 공급부족 피크로 전망**(기존 "공급부족 최소 2028년까지"
  판정과 함께, 2027년이 그 안에서도 정점 구간이라는 세부 시점 확보). PC/모바일은
  상대적으로 수요 약화 신호(CY27 할당량 25~35%+ 축소 가능성) — 서버·HBM
  중심으로 공급이 재배분되는 구도.

#### 3-1. 추적 프레임 — "경쟁 생존 논리" vs "ROIC 규율" (2026-07-23 사용자 논의)

사용자와의 논의에서 정리된 해석 프레임. 알파벳이 주가 급락을 실시간으로 보면서도
CapEx 축소 신호를 전혀 주지 않은 것(오히려 2027년 "significantly increase"
재확인)은, **"이번 분기 주주 반응"보다 "AI 인프라 경쟁에서 밀리면 회사
존재 자체가 위태로워진다"는 경영진 판단이 더 무겁다**는 뜻으로 해석할 수
있음. 게임이론적으로 보면 하이퍼스케일러 입장에서는 과잉투자의 리스크(ROI
미달)보다 과소투자의 리스크(경쟁 탈락·클라우드 점유율 영구 상실)가 더
크다고 판단하는 구도.

**SK하이닉스 함의**: 이 논리가 유효하다면, 하이퍼스케일러 주가가 흔들려도
메모리 주문 자체는 줄어들지 않는다 — 경쟁 생존을 위한 투자는 주주 눈치
때문에 꺾이지 않기 때문. 7/23 08시 프리마켓에서 SK하이닉스가 알파벳
주가 하락에도 불구하고 갭업(1,880,000원, +2.73%)한 것을, 시장이 이 논리를
선반영하기 시작한 신호로 잠정 해석([현재 상태 참고](../entities/sk-hynix.md)).

**균열 가능 지점(버핏 ROIC 프레임과의 긴장)**: 이 "경쟁 생존" 논리가
무한정 통하는 건 아님 — 과거 메타의 메타버스 투자가 이사회·행동주의
투자자 압박으로 축소된 전례처럼, 수익화 지연이 장기화되면 결국 ROIC
규율이 개입할 수 있음([ROIC를 투자의 핵심 기준으로 보기](roic-as-investment-criterion.md)
참고). "경쟁에서 이겨야 존재의미가 있다"는 논리와 "투자한 돈을 결국
회수해야 한다"는 논리가 정면으로 부딪히는 구간.

**다음 검증 지점(계속 추적)**:
1. **7/29 메타·마이크로소프트, 7/30 아마존 실적** — CapEx 추가 상향에도
   주가가 비슷하게 하락하는 패턴이 반복되는지(반복되면 "경쟁 생존 논리"가
   업계 전반의 컨센서스라는 뜻, 반복 안 되면 알파벳 특이 사례로 축소 해석)
2. **7/29 SK하이닉스 자체 실적** — 매출·영업이익이 컨센서스(84.57조/64.80조)를
   상회하며 실제로 가속하는지, 이 논리의 최종 펀더멘털 검증대
3. **하이퍼스케일러들의 후속 CapEx 축소·둔화 신호 여부** — ROIC 규율이
   개입하는 신호(감가상각 부담·잉여 인프라 매각·데이터센터 계획 축소 등)가
   나타나는지 매 체크마다 긍정+부정 키워드로 계속 감시

### 4. LTA(장기공급계약) 구조
- 기준: 3~5년 확대·가격하단 강화(긍정) / 기존 유지(중립) / 계약축소·가격하단
  약화(훼손)
- 지지: 한국투자증권(LTA로 실적 변동성 축소, 멀티플 리레이팅 근거), SK증권
- **(2026-07-17 아침 체크) 긍정 강화 — 가격 상한선 없음**: SK하이닉스가 최근
  체결한 LTA에는 **가격 상한선을 두지 않음**이 확인됨 — 마이크론 등
  경쟁사는 장기계약에 상한선을 설정하는 것과 대비되는 유일한 구조
  (TrendForce, 뉴데일리 보도). 가격이 계속 오를 경우 이익 상방이 경쟁사보다
  더 크게 열려있다는 의미로, 기존 "LTA 확대"라는 컨센서스를 한 단계 더
  구체화하는 강한 긍정 디테일.
- **(2026-07-23 낮 인제스트) floor/ceiling 구조 디테일**: LTA/SCA 장기계약에
  floor/ceiling 가격이 포함돼 있으나, **비-LTA(스팟) 고객의 ASP 상승 기대는
  제한하지 않음** — 즉 LTA로 기본 수익을 확보하면서도 스팟·단기 계약에서는
  추가 프리미엄을 그대로 수취 가능한 구조. "가격 상한선 없음" 발견을
  한 단계 더 구체화 — 장기계약이 상방을 막지 않으면서 하방만 방어하는
  "논-다운사이드" 구조에 가깝다는 해석.

### 6. ADR 프리미엄 — 위 "해결" 항목과 직결
- 기준: 10~25% 안정(긍정) / 25~35% 높음(중립) / **35%+ 과열(훼손)**
- **2026-07-16 계산 완료: 약 30.7%("높음·중립"), 정점 51.5%에서 narrowing 중**
- 강세 해석: 대신증권·NH(ADR 상장 = 밸류에이션 할인 해소·리레이팅 계기) —
  실제로 프리미엄이 좁혀지고 있어 이 해석이 힘을 받는 쪽
- **약세 해석: BNK("ADR은 거래 편의성일 뿐 밸류에이션 변화 요인 아님"),
  Morningstar(원웨이 fungibility로 프리미엄·변동성만 확대)** — 즉 같은
  "ADR 프리미엄"이라는 현상을 두고 강세론은 "수렴할 기회"로, 약세론은
  "구조적으로 계속 벌어질 위험"으로 정반대로 해석했으나, 지금까지의 실제
  추이(51.5%→30.7%)는 강세 쪽 해석과 더 부합한다.
- **(2026-07-16 저녁 체크) UBS의 상대가치 트레이드**: UBS는 목표가 320만원·
  매수 의견을 유지하면서도 **"ADR은 매수, 본주는 매도"**라는 페어 트레이드를
  권고 — 이는 강세도 약세도 아닌 제3의 해석: "괴리율 자체가 방향성 베팅
  대상"이라는 시각. ADR 프리미엄이 narrowing 중이라면 이 페어 트레이드는
  ADR 상대 강세(또는 본주 상대 약세)에 베팅하는 셈이라, 우리가 확인한
  30.7%·narrowing 흐름과 정합적인지는 계속 지켜볼 필요.
- **(2026-07-20 아침 체크) 안정 구간 진입 — 약 21.6%**: 날짜 동기화 페어
  (본주 7/16 종가 1,859,500원 ↔ ADR 7/16 미국 세션 종가 $152.31, 환율
  1,484.7원 — 7/15 확정치 근사)로 재계산한 프리미엄은 **약 21.6%**,
  판정 기준상 처음으로 **"10~25% 안정(긍정)" 구간**에 들어왔다. 정점
  51.5% → 30.7% → 21.6%. ADR이 7/16 미국 세션에서 -13.7%로 본주(-10.69%)
  보다 더 크게 빠지며 괴리가 좁혀진 것. tradingkey도 "프리미엄 급격히
  축소, 양방향 전환 임박" 보도. ⚠ 환율 근사로 ±1~2%p 오차 가능, UBS
  페어트레이드(ADR 매수·본주 매도) 관점에서는 **불리하게 움직인 셈**.

  약세·차익실현·HBM4 축소 재인용·금리 경계감)에 이은 다섯 번째 요인.

## 투자 프레임: Buffett의 ROIC 기준과의 정렬 (2026-07-20 신규)

Warren Buffett의 최근 AI 인터뷰에서 "AI 시대에도 결국 좋은 기업은 변하지 않는다"는
메시지가 나왔다. 이는 현재의 9개 체크포인트 분석과 일관성을 갖는다:

- **Buffett의 핵심**: "국채보다 훨씬 높은 수익을 장기간 낼 수 있는 기업" = **ROIC 유지**
- **우리의 체크포인트와의 교집합**:
  - ③(AI CapEx): "CapEx는 증가하는데 이익은 따라갈 것인가?" = **자본효율성 문제**
  - ②(HBM 점유율): SK하이닉스의 HBM 마진은 경쟁사보다 높은가? = **ROIC 우위**
  - ④(LTA): 가격 상한선 없음 = 수익 상방이 더 크다 = **ROIC 상향 여력**
  - ⑨(실적): 영업이익 성장률(+650% YoY)이 지속 가능한가? = **ROIC 지속성**

**핵심 질문 재정의**:
- "SK하이닉스의 주가가 오를까?"가 아니라
- "SK하이닉스가 AI 시대에도 높은 자본수익률(ROIC)을 유지할 수 있는가?"

자세한 내용은 [ROIC를 투자의 핵심 기준으로 보기](roic-as-investment-criterion.md) 참고.
