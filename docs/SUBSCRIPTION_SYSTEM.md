# 청약 준비 점수화 시스템
**Date**: 2026-07-16  
**Purpose**: 국민주택 청약 경쟁력 정량화 및 준비도 추적

---

## 1. 시스템 개요

청약 시스템은 **자격 확인 → 경쟁력 점수 → 준비도 추적 → 의사결정 지원** 의 4단계로 구성됩니다.

```
청약 자격 확인 (Eligibility Check)
    ↓
경쟁력 점수 (Competitiveness Score)
    ↓
준비도 추적 (Readiness Tracking)
    ↓
의사결정 지원 (Decision Support)
```

---

## 2. 청약 자격 확인 (Eligibility Check)

### 2.1 필수 자격 요건

| 항목 | 기준 | 현재 상태 |
|------|------|---------|
| 청약통장 가입 | 2년 이상 | ✅ 20년 8개월 |
| 납입 회차 | 최소 12회 이상 | ✅ 249회 |
| 거주 지역 | 서울/경기/지방 | ✅ 경기도 (용인) |
| 세대주 여부 | 세대주 필수 | 확인 필요 |
| 기존 주택 소유 | 무주택자 필수 | 확인 필요 |
| 기혼/미혼 | 제한 없음 | 확인 필요 |

### 2.2 청약 순위 결정 요소

**1순위 (가장 경쟁력 높음)**
- 청약통장 2년 이상 + 납입 조건 충족
- 세대주 무주택자
- 해당 지역 거주 또는 직장 위치

**2순위**
- 청약통장 미보유 또는 미달
- 기존 주택 소유 후 매각

## 3. 경쟁력 점수 (Competitiveness Score)

### 3.1 점수 산정 방식

총점 100점 = 청약통장 점수(40) + 거주/직장 점수(30) + 자산 점수(20) + 준비도 점수(10)

### 3.2 세부 점수 계산

#### A. 청약통장 점수 (만점 40점)

**가입 기간 (0~15점)**
```
- 20년 이상: 15점
- 10~20년: 12점  ← 현재: 20년 8개월 = 15점
- 5~10년: 8점
- 2~5년: 4점
- 2년 미만: 0점
```

**납입 회차 (0~15점)**
```
- 240회 이상: 15점  ← 현재: 249회 = 15점
- 180~240회: 12점
- 120~180회: 8점
- 24~120회: 4점
- 12~24회: 2점
- 12회 미만: 0점
```

**납입액 수준 (0~10점)**
```
- 최고액(250,000원) 납입 중: 10점  ← 현재 = 10점
- 고액(200,000~250,000원): 8점
- 중액(150,000~200,000원): 5점
- 저액(100,000~150,000원): 2점
- 최저액(100,000원 이하): 0점
```

**청약통장 점수 = 15 + 15 + 10 = 40점 (만점)**

#### B. 거주/직장 점수 (만점 30점)

**거주지/직장지 우선순위 (0~20점)**
```
- 목표 지역 거주: 20점
- 인접 지역 거주: 15점
- 목표 지역 직장: 12점  ← 현재 상황 확인 필요
- 인접 지역 직장: 8점
- 서울 거주: 10점
- 경기도 거주: 5점  ← 현재 = 5점
- 지방 거주: 2점
```

**세대주 여부 (0~10점)**
```
- 세대주 무주택자: 10점  ← 확인 필요
- 세대주 주택 소유: 5점
- 세대원: 2점
```

#### C. 자산 점수 (만점 20점)

**청약금 준비 상황 (0~12점)**
```
- 목표금액 이상 (≥5억원): 12점
- 목표금액 80% 이상: 10점
- 목표금액 60% 이상: 8점  ← 현재: 28,050,000원 ~ 조건 확인
- 목표금액 40% 이상: 5점
- 목표금액 미만: 2점
```

**전세/월세 계획 (0~8점)**
```
- 청약금 + 임차금 충분: 8점  ← 현재 상황 확인 필요
- 청약금 충분, 임차금 준비중: 6점
- 청약금만 충분: 4점
- 청약금 준비중: 2점
- 미계획: 0점
```

#### D. 준비도 점수 (만점 10점)

**청약 대기 상태 (0~5점)**
```
- 공고 확인 중 + 준비 완료: 5점
- 공고 확인 중: 3점
- 확인 미흡: 1점
```

**서류 준비 (0~5점)**
```
- 필요 서류 100% 준비: 5점
- 80% 이상 준비: 3점
- 50% 이상 준비: 2점
- 미준비: 0점
```

## 6. 의사결정 지원 (Decision Support)

### 6.1 청약 추천 점수 기준

| 점수 | 권장 사항 |
|------|---------|
| 80점 이상 | ✅ 공고 출현 시 즉시 청약 |
| 70~80점 | 🟡 조건 재검토 후 청약 |
| 60~70점 | 🟡 경쟁율 확인 후 선택 |
| 50~60점 | ⚠️ 준비 강화 후 다음 공고 대기 |
| 50점 미만 | ❌ 자격 확인 또는 준비 연장 |

### 6.2 현재 의사결정 기준

사용자의 **청약통장 점수 40점(만점)**은 매우 우수하므로:

1. **공고 출현 시**: 즉시 신청하기 (경쟁력 상위 10% 수준)
2. **임차금 계획**: 전세 만료 전 마크업으로 자금 확보
3. **지역 전략**: 용인 플랫폼시티 우선, 인접 지역 검토

## 7. 청약금 계획 (Subscription Fund Planning)

### 7.1 목표금액 산정

**경기도 용인 플랫폼시티 기준** (가정)

```
예상 분양가: 5억원 (공급 규모에 따라 변동)
청약금(10%): 50,000,000원
현재 잔액: 28,050,000원
부족분: 21,950,000원

월 추가 저축(200만원 기준): 약 11개월

→ 2026-11월까지 충분히 확보 가능
```

### 7.2 자금 조달 경로

1. **청약통장**: 28,050,000원 (기본)
2. **추가 저축**: 월 200~250만원 × 10개월 = ~25,000,000원
3. **전세금 반환**: 기존 전세금 마크업 → 임차금 + 추가 청약금
4. **긴급 자금**: 퇴직연금 (필요시, 사용자 의향 확인)

**목표**: 2026-11월까지 청약금 50,000,000원 이상 확보

## 10. 주의사항 (Cautions)

1. **공고 즉시성**: 공고 출현 시 빠른 의사결정 필요 (선착순 마감 가능)
2. **자격 유지**: 청약 접수 전까지 세대주/무주택 자격 유지 필수
3. **자금 확보**: 목표금액 미달 시 참여 불가 (청약 철회 불가)
4. **지역 우선**: 용인 플랫폼시티 공고가 나올 때까지 인접 지역 검토
5. **세제 혜택**: 생애 최초 주택 구입 시 양도세/취득세 감면 확인
