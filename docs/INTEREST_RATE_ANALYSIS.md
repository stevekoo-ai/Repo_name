## 2. 금리 평가 로직 (Interest Rate Scoring)

### 2.1 평가 프레임워크

총점 100점 = 절대 금리 평가(30) + 추이 분석(30) + 양국 차이(25) + 시장 신호(15)

### 2.2 세부 평가 기준

#### A. 절대 금리 수준 (만점 30점)

**미국 10Y Treasury 평가**
```
0.5% 이하:       0점  (매우 낮음, 경기 약세)
0.5% ~ 1.5%:    10점  (낮음, 완화적)
1.5% ~ 2.5%:    15점  (중간, 중립적)  ← 현재 기준선
2.5% ~ 3.5%:    20점  (보통, 타이트)
3.5% ~ 4.5%:    25점  (높음, 긴축적)
4.5% 이상:      30점  (매우 높음, 강한 긴축)
```

**한국 10Y 국고채 평가**
```
1.5% 이하:       0점  (매우 낮음)
1.5% ~ 2.0%:    10점  (낮음)
2.0% ~ 2.5%:    15점  (중간)  ← 현재 기준선
2.5% ~ 3.0%:    20점  (보통)
3.0% ~ 3.5%:    25점  (높음)
3.5% 이상:      30점  (매우 높음)
```

**절대 금리 점수 = (US 점수 + KR 점수) / 2**

#### B. 추이 분석 (만점 30점)

**1개월 변화율 (0~15점)**
```
US 10Y 상승 추세:
- 상승 0~50bp:       5점   (완만한 상승)
- 상승 50~100bp:    10점   (가파른 상승)
- 상승 100bp 이상:  15점   (급격한 상승)

US 10Y 하강 추세:
- 하강 0~50bp:       5점   (완만한 하강)
- 하강 50~100bp:    10점   (가파른 하강)
- 하강 100bp 이상:  15점   (급격한 하강)

KR 10Y와의 괴리:
- 동일 추세:        0점   (정상)
- 반대 추세:        -5점  (비정상, 주의)
```

**3개월 추세 (0~15점)**
```
양국 모두 상승:     10점  (긴축 사이클)
양국 모두 하강:     10점  (완화 사이클)
미국 리드:          12점  (미국이 선행)
한국 리드:           8점  (한국이 독립적)
반대 추세:           5점  (비정상, 조정 대기)
```

**추이 점수 = 1M 변화 점수 + 3M 추세 점수**

#### C. 양국 금리 차이 (만점 25점)

**Yield Spread (US 10Y - KR 10Y)**

```
평균 스프레드: 200~250bp (역사적 평균)

Spread 수준에 따른 점수:
- 100bp 미만:        0점   (한국 금리 높음, 비정상)
- 100~150bp:         5점   (한국 금리 상대적 높음)
- 150~200bp:        10점   (한국 금리 중간 높음)
- 200~250bp:        15점   (정상 범위, 기준)  ← 목표
- 250~300bp:        15점   (정상 범위, 기준)
- 300~350bp:        10점   (한국 금리 상대적 낮음)
- 350bp 이상:        5점   (한국 금리 매우 낮음)
```

**Spread 변화 추이 (0~10점)**
```
Spread 확대 추세:
- 0~25bp 확대:       5점   (완만한 확대)
- 25~50bp 확대:      8점   (중간 확대)
- 50bp 이상 확대:   10점   (급격한 확대)

Spread 축소 추세:
- 0~25bp 축소:       5점   (완만한 축소)
- 25~50bp 축소:      8점   (중간 축소)
- 50bp 이상 축소:   10점   (급격한 축소)

Spread 안정:
- ±10bp 이내:        3점   (매우 안정적)
```

**Spread 점수 = 절대 수준 점수 + 변화 추이 점수**

#### D. 시장 신호 (만점 15점)

**역수익 곡선 신호 (Recession Signal)**
```
US 10Y-2Y Spread:
- 양수 (Positive):   10점   (정상, 경기 순환)
- -0.25 ~ 0:         8점   (경고 신호)
- -0.25 ~ -0.5:      5점   (역수익 시작)
- -0.5 이하:         0점   (깊은 역수익, 경기 약세)

KR에는 직접 역수익 곡선이 없으므로:
- 10Y-3Y Spread로 대체

양국 신호 일치도:
- 일치:              5점   (신뢰도 높음)
- 불일치:           2점   (신호 모호)
```

**금융 조건 경직도 (Financial Conditions Tightness)**
```
금리 상승 + Spread 확대:  10점  (경직)
금리 상승 + Spread 축소:   5점  (중간)
금리 하강 + Spread 확대:   3점  (완화)
금리 하강 + Spread 축소:   1점  (극도 완화)
```

**시장 신호 점수 = 역수익 신호 + 금융 조건**

## 4. 포트폴리오 전략 연계

### 4.1 금리 점수에 따른 자산 배분 전략

| 점수대 | 상황 | 채권 | 주식 | 환율(USD) | 액션 |
|--------|------|------|------|----------|------|
| 85~100 | 극도 완화 | +40% | -10% | Weak | 주식 집중 |
| 70~85 | 완화 | +30% | +10% | Weak | 주식 중심 |
| 55~70 | 중간 | +20% | +20% | Neutral | 균형 배분 | ← 현재 |
| 40~55 | 긴축 | +10% | +30% | Strong | 신중한 주식 |
| 0~40 | 극도 긴축 | 0% | -20% | Strong | 현금/채권 |

**현재 권장**: 균형 배분 (주식 50% + 채금 30% + 현금 20%)

### 4.2 금리 시나리오별 전략

#### 시나리오 A: 미국 금리 급락 (4.2% → 3.0%)

```
확률: 20% (경기 약세 신호 시)
영향:
  - Spread 축소 (140bp → 100bp)
  - 한국 금리는 3.0% → 2.5% (완화)
  - 달러 약세
  
포트폴리오 영향:
  - 채권: +8% (채권 가격 상승)
  - 주식: -5% (경기 약세)
  - USD/KRW: 1,150 → 1,120 (약세)
  
권장 액션: 채권 비중 +10%
```

#### 시나리오 B: 미국 금리 유지, 한국 금리 인상 (3.0% → 3.2%)

```
확률: 50% (현재 가장 가능성 높음)
영향:
  - Spread 축소 (140bp → 120bp)
  - 한국 긴축 신호
  - USD/KRW 약세 완화
  
포트폴리오 영향:
  - 채권: -3% (한국 채권 약세)
  - 주식: +2% (현상 유지)
  - USD/KRW: 1,150 → 1,170 (약세)
  
권장 액션: 현상 유지, 한국 채권 헤지
```

#### 시나리오 C: 양국 금리 상승 (인플레 심화)

```
확률: 30% (inflation resurgence)
영향:
  - US 10Y: 4.2% → 4.8%
  - KR 10Y: 2.8% → 3.3%
  - Spread 유지 또는 확대
  
포트폴리오 영향:
  - 채권: -10% (채권 가격 하락)
  - 주식: -8% (고금리 영향)
  - USD/KRW: 1,150 → 1,200 (강세)
  
권장 액션: 채권 헤지, 달러 헤지 축소
```

## 5. 의사결정 기준 (Decision Framework)

### 5.1 포트폴리오 리밸런싱 트리거

**트리거 1: 금리 점수 20점 이상 변화**
```
현재: 68점
상한: 88점 → 채권 비중 +5%
하한: 48점 → 주식 비중 +5%
```

**트리거 2: Spread 50bp 이상 변화**
```
현재: 140bp
확대 → 190bp 이상: 한국 채권 헤지
축소 → 100bp 이하: 한국 채권 비중 +5%
```

**트리거 3: 역수익 곡선 진입 (US 10Y-2Y < 0)**
```
발생 시: 주식 비중 -10%, 현금 +10%
해제 시: 현상 회복
```

### 5.2 투자 타이밍 신호

#### 좋은 진입 타이밍 (매수 신호)
```
✅ 금리 점수 < 50점 (완화 극심)
✅ 채권 수익률 > 5% (충분한 리턴)
✅ Spread > 250bp (기회 국면)
✅ 역수익 해제 + 금리 상승 초기
```

#### 진입 조심 (중립 신호)
```
⏳ 금리 점수 50~70점 (현재 상황)
⏳ Spread 140~180bp (평균 근처)
⏳ 금리 추이 불명확
```

#### 조심스러운 진입 (매도 신호)
```
⚠️ 금리 점수 > 80점 (긴축 극심)
⚠️ 역수익 곡선 (경기 약세)
⚠️ 금리 급등 추세
⚠️ Spread 축소 + 한국 금리 유지
```

## 6. SK Hynix에 미치는 영향 분석

### 6.1 금리와 SK Hynix의 연관성

```
높은 금리 환경:
  - 긍정 요인: 달러 강세 (SK Hynix는 달러 수익)
  - 부정 요인: 반도체 산업 경기 약화
  - 순효과: 중간 (변동성 높음)

금리 하락 환경:
  - 긍정 요인: 반도체 경기 회복 + 기술주 강세
  - 부정 요인: 달러 약세 (달러 수익 감소)
  - 순효과: 긍정적 (성장성 높음)
```

### 6.2 현재 금리 환경에서 SK Hynix 전망

**금리 점수 68점 → "중간 수준 긍정적"**

```
금리 요인: ⏳ 중립
  - 미국: 여전히 높음 (4.2%) → 달러 강세 유리
  - 한국: 중간 (2.8%) → 부정 요인 없음

종합 평가:
  - 단기(3개월): 금리 안정 + AI 호재 → 상승 가능성 60%

  - 장기(12개월): 금리 인하 주기 전환 → 주도주 회전
```

### 6.3 모니터링 지표

```
매주 확인:
  - US 10Y vs KR 10Y (Spread 추이)
  - US 역수익 곡선 여부
  - 달러 인덱스 (SK Hynix 환리스크)

매월 확인:
  - Fed 금리 결정 (긴축/완화 신호)
  - 한국은행 기준금리 (경기 전망)
```
